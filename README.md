I'm a self-taught iOS Developer with a deep passion for crafting polished mobile experiences and UI/UX design. I enjoy breaking down complex concepts into clean, well-structured code — and sharing what I learn with the community through open source projects and educational content.

- 🔨 Currently building **[Linkwise](https://linkwise.app)** — an AI-powered personal knowledge system for iOS that transforms saved web content into a searchable, queryable collection
- 📱 Sharing iOS tips, dev insights, and behind-the-scenes content on [Instagram](https://instagram.com/dheeraj.iosdev)
- 🌱 Always exploring the intersection of native iOS development, AI integration, and thoughtful product design
